About Me

I enjoy turning ideas into clear page structures with headings, paragraphs, links, images, and semantic sections.

Right now I am focusing on writing valid HTML, improving accessibility, and understanding how browsers read document structure.

"A good webpage is easy to read, easy to navigate, and easy to understand."

Learning Goals

This Month

  1. Build semantic page layouts with header, main, section, and footer.
  2. Use accessible links and images with clear link text and useful alt descriptions.
  3. Create lists and tables that present information in a clean, readable way.

Topics I Am Practicing

Semantic HTML
Using meaningful elements so both people and assistive technologies understand the page.
Accessible Navigation
Creating internal links and page landmarks that help users move through content quickly.
Content Structure
Choosing between paragraphs, lists, tables, and figures based on the type of information.
