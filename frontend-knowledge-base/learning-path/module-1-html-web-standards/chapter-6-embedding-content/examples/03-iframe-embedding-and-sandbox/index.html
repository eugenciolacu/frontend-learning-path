Iframe Embedding and Sandbox

This example embeds a local HTML document inside an iframe. The iframe uses a restricted sandbox attribute and lazy loading to demonstrate safer embedding.

Sandboxed Local Embed

Example Markup for External Embeds

<iframe
	src="https://www.youtube.com/embed/example-id"
	title="YouTube video player"
	loading="lazy"
	allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
	allowfullscreen
></iframe>
