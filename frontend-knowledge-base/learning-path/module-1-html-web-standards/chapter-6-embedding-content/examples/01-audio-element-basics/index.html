Audio Element Basics

This example shows a browser audio player with multiple sources, fallback text, and a transcript block. The example uses public sample files so the markup works without storing binary media in the repository.

Lesson Introduction Audio

Use the controls to play the audio clip. The browser will pick the first supported source.

Key Markup

<audio controls preload="metadata">
	<source src="lesson.ogg" type="audio/ogg">
	<source src="lesson.mp3" type="audio/mpeg">
	Your browser does not support HTML audio.
</audio>

Transcript

"Welcome to the embedding content lesson. In HTML, audio can be embedded with the audio element and one or more source elements for compatibility."
