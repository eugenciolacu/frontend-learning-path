Video Element and Captions

This example demonstrates multiple video sources, a poster image, and a caption track. If the network is unavailable, the fallback text inside the player still explains what the markup does.

HTML Video Player

Open the captions menu in the player to enable the English caption track.

Key Markup

<video controls preload="metadata" poster="poster.svg">
	<source src="lesson.webm" type="video/webm">
	<source src="lesson.mp4" type="video/mp4">
	<track kind="captions" src="captions-en.vtt" srclang="en" label="English" default>
	Your browser does not support HTML video.
</video>
