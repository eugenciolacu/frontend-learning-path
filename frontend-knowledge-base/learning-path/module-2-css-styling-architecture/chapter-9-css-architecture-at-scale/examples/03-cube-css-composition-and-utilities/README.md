# Example 3: CUBE CSS Composition and Utilities

## Purpose
This example shows how CUBE CSS separates composition classes from block styles.

## What to Notice
- `wrapper`, `stack-sm`, `stack-lg`, and `cluster` are composition classes.
- `lesson-card` is the block.
- `lesson-card--accent` acts as an exception or variation.
- The layout spacing is controlled outside the component, which keeps the block easier to reuse.

## How to Use
Open `index.html` and compare how spacing and flow are controlled by composition classes rather than page-specific selectors.
