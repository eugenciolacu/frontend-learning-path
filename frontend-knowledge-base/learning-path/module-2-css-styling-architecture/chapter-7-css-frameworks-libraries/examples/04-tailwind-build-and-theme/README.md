# Example 4: Tailwind Build and Theme Configuration

This example demonstrates:
- a build-based Tailwind setup instead of the browser CDN
- adding project theme tokens in `tailwind.config.js`
- using a source CSS file and a generated output CSS file
- creating a reusable small component layer with `@layer components`

Suggested steps:
1. Run `npm install` inside this example folder.
2. Run `npm run build` to generate `src/output.css`.
3. Open `src/index.html` in a browser.

If you want live rebuilding while editing, run `npm run dev`.
