# Example 6: CSS Functions and Modern Color Spaces

This example demonstrates:
- `clamp()` for fluid typography
- `min()` and `max()` for safer sizing and spacing limits
- `calc()` combined with custom properties
- fallback colors followed by `lch()` and `oklab()` declarations

Open `index.html` and resize the viewport. Watch the heading, spacing, and panel width adapt smoothly. The color declarations include safe fallbacks for browsers with partial support.
