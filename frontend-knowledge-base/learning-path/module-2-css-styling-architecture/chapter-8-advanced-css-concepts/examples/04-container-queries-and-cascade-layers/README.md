# Example 4: Container Queries and Cascade Layers

This example demonstrates:
- `container-type: inline-size` on a wrapper
- `@container` rules that respond to the component's own width
- `@layer` used to separate reset, tokens, components, and utilities
- the same component adapting differently in narrow and wide containers

Open `index.html` and compare the cards in the narrow and wide columns. The viewport stays the same, but the component layout changes because its containers are different.
