Float and Clear

Floated box

Floats were originally designed so content such as text could wrap around media. In this example, the colored box is floated to the left side of the article.

The paragraphs flow around the floated element instead of starting below it. This was a very common technique in older web layouts before Flexbox and Grid became the preferred tools for major page structure.

Modern developers still need to understand floats because they appear in legacy code and can still be useful for article-like content.

This footer uses clear: both and starts below the float.
