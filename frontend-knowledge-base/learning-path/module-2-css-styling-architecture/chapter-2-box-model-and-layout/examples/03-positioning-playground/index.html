Sticky

I stick to the top of this scrolling area.

Sticky positioning is useful for local navigation, labels, and headings.

Scroll this section to watch the note pin itself when it reaches the top.

Because the section is tall, there is enough space for the sticky behavior to be visible.

Browsers keep the element in normal flow until the threshold is reached.

After that, the element behaves like it is attached to the scroll container edge.

Sticky elements still need an offset such as top: 0.

This makes them easier to reason about than fixed elements in many layouts.

They are often used for table headers and side navigation.
