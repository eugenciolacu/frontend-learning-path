Overflow and Z-Index

Overflow Auto

Overflow lets a box stay within a defined size while preserving access to content.

This is useful for logs, code samples, chat panels, or any area that can grow.

When content becomes taller than the panel, the browser adds scrolling.

Using overflow: auto keeps the layout stable.

Without it, content may spill outside the box and overlap neighboring UI.

That behavior is sometimes useful, but often it is a bug.
