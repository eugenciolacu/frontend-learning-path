# Example 4: Grid Template Areas Dashboard

This example demonstrates:
- a page shell built with `grid-template-areas`
- named sections for header, sidebar, main content, and footer
- a responsive layout that rearranges areas on small screens
- why Grid is useful for larger structural layouts

Open `index.html` and resize the viewport. Notice how the same HTML is rearranged by changing the grid areas at the breakpoint.
