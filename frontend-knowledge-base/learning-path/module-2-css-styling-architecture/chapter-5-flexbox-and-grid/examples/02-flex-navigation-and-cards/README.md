# Example 2: Flex Navigation and Cards

This example demonstrates:
- a navigation bar aligned with Flexbox
- a wrapped card row using `flex: 1 1 16rem`
- how `flex-basis` helps define a minimum comfortable card width
- how component internals can use a column flex layout

Open `index.html` and resize the page. Notice that the navigation links wrap and the cards reorganize themselves without changing the HTML structure.
