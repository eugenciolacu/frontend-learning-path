# Example 1: Flex Axes and Alignment

This example demonstrates:
- a flex container with a row main axis
- the difference between `justify-content` and `align-items`
- wrapping cards with `flex-wrap`
- clean spacing with `gap`

Open `index.html` in a browser and resize the window. Observe how the top toolbar aligns items on both axes and how the cards wrap when horizontal space becomes limited.
