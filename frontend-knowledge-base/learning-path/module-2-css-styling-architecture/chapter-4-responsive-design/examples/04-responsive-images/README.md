# Example 4: Responsive Images

This example demonstrates:
- the standard `img` responsive rule with `max-width: 100%`
- a `<picture>` element for art direction
- different image compositions for narrow and wide layouts
- accessible image markup with `alt` text and captions

Resize the browser and inspect the `<picture>` element. A different SVG file is used when the viewport becomes wider, which is useful when the same subject needs a different crop or composition.
