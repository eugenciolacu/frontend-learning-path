# Example 3: Min and Max Width or Height

This example demonstrates:
- `max-width` to control line length and component growth
- `min-width` for a media-query-based layout change
- `max-height` to reduce vertical spacing on short screens
- `min-height` to center content on taller screens

Try both resizing the browser to be narrow and making it shorter in height. This is useful for understanding that responsive design is not only about width.
