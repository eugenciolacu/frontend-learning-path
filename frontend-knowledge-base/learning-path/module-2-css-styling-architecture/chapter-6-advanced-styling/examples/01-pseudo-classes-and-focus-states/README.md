# Example 1: Pseudo-Classes and Focus States

This example demonstrates:
- interactive pseudo-classes such as `:hover`, `:active`, and `:focus-visible`
- container highlighting with `:focus-within`
- structural styling with `:nth-child(odd)`
- accessible focus feedback for keyboard users

Open `index.html` in a browser. Hover the button, press it, tab through the controls, and notice how the search panel reacts when the input receives focus.
