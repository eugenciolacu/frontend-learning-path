# Example 6: Shadows, Filters, Border Radius, and Clip Path

This example demonstrates:
- layered `box-shadow` values for depth
- `filter` effects such as blur, saturate, and grayscale
- controlled corner shaping with `border-radius`
- a decorative ribbon built with `clip-path`

Open `index.html` in a browser and compare how the different cards create hierarchy with depth, shape, and image treatment.
