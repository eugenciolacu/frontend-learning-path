# Example 2: Pseudo-Elements and Decorative Details

This example demonstrates:
- decorative content with `::before` and `::after`
- a drop cap with `::first-letter`
- highlighted opening text with `::first-line`
- how pseudo-elements can improve presentation without extra wrapper elements

Open `index.html` in a browser and inspect the quote card, the editorial paragraph, and the status badge. Notice that several visible details are created entirely in CSS.
